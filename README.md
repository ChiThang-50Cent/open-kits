# OpenCode Kits

Custom tools for opencode/claude code.

## Tools

### [ddgs-search](./ddgs-search/)

Hybrid web search helper combining DDGS and SearXNG. Available as:
- OpenCode custom tool via `ddgs-search.ts`
- Claude Code MCP server via `server.py`

See [ddgs-search/README.md](./ddgs-search/README.md) for installation and usage.
